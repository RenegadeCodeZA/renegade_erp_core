# Renegade ERP Core

A comprehensive ERPNext extension module that provides core functionality for customized ERP solutions.

## Features

### Planned Modules:
- **Whitelabeling System** - Custom branding and UI customization
- **Middleware Configuration** - API and integration settings
- **AI Integration** - AI keys and configuration management
- **Core Extensions** - Base functionality extensions for ERPNext

### Installation

This is an ERPNext application. Install it using:

```bash
bench get-app renegade_erp_core
bench install-app renegade_erp_core
```

### Development

This application is designed with a modular architecture to support different functional areas through organized menu sections and configurations.
